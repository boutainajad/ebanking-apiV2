# NexaAuth SPA - React.js Application

Application d'authentification avec React.js et Laravel API supportant connexion classique et OAuth2 (Google + GitHub).

## Technologies

- **Frontend:** React 18, TypeScript, Vite, Tailwind CSS, Zustand, React Router
- **Backend:** Laravel 10, Sanctum, Socialite

## Installation Frontend

```bash
# Cloner le projet
git clone https://github.com/yourusername/nexa-auth-spa-react.git
cd nexa-auth-spa-react

# Installer les dépendances
npm install

# Configurer l'environnement
cp .env.example .env

# Lancer l'application
npm run dev
